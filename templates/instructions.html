The list below provide instructions for the Type Ten game.

Main Menu

The main menu provides options to start a new game, view the instruction page, access the leaderboard or log in and out of the game. To login and play the game a Goolge account is required.

New Game

To start a new game, simply select the option from the main menu. Every game has ten rounds. Each round has one question that requires 10 answers, you have 60 seconds to find as many answers as possible.

Score

For each question there are answers worth 10 points, 15 points and 20 points according do the difficulty. Every answer you find will be added to your score.

LeaderBoard

While your score increases the leaderboard will adjust your ranking to show if you have manage to reach the top 10 players.

Correct Answers

Each time you find a correct answer it will be placed on the right hand side so you can keep an eye on what you have already found and the points you have gained.

Wrong Attempts

Wrong attempts will be placed on the middle and there is no limit on how much you can try to get the correct answers.

Timer

Each round you get 60 seconds to type as many answers as possible, in the last few seconds the timer will get red to alarm you that time is running out, so type faster !. Once the time runs out you will get 10 seconds to prepare for the new round.
